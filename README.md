# 医学教材文本切分方法对比实验
**面向医学RAG系统的文本切分方案对比 **

---

## 📌 项目简介
本项目针对**内外科学医学文本**（知识点长、术语密集、逻辑链长、跨段落关联强）的特点，开展**4种主流文本切分方法**的量化对比实验，验证自研 **RecursiveMaxMin混合切分算法** 在复杂医学问答检索任务上的效果，为医学领域RAG切分方案的选择提供参考。

## 🧪 实验背景
在检索增强生成（RAG）流程中，文本切分直接决定检索召回率与问答准确性。
通常认为语义切分耗时最久但切分效果最好，然而实验对比发现在多个方面递归切分都优于语义切分，这引起了我的兴趣。
在
本实验通过统一评估框架，对比不同切分策略的优劣。

## 📊 实验数据
- **文本**：《外科学》完整版教材（神经外科/普外科/骨科/胸外科等）
- **问答集**：100条高质量外科学问答对
  - 事实型、推理型、跨段落型、边界型
  - 覆盖定义、病因、体征、手术指征、鉴别诊断

## 🛠️ 对比切分方法
1. **固定长度切分**：按固定字符硬切分
2. **递归结构切分**：按段落/句子边界分层切分
3. **传统语义切分**：基于句子相似度合并
4. **自研 RecursiveMaxMin 混合切分**
    - 递归粗切分 → 语义合并 → 长度归一化
    - 适配外科长知识点 + 不截断句子 + 长度合规

## 📏 评估指标
### 核心检索指标
- **HitRate@3**：Top3 检索结果包含标准答案的比例（越高越好）
- **MRR**：正确答案平均排序分数（越高越好）

### 文本结构指标
- 块数量、平均长度、完整句比例、语义连贯性、碎片数量

## 📈 实验结果
| 切分方法 | HitRate@3 | MRR | 块数量 | 平均长度 | 完整句比例 | 语义连贯性 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |

| 递归切分 | 0.3700 | 0.3067 | 128 | 477.9 | 0.344 | 0.823 |
| 传统语义切分 | 0.2700 | 0.2117 | 735 | 126.0 | 0.412 | 0.785 |
| **自研MaxMin混合切分** | **0.4200** | **0.3100** | **112** | **452.3** | **0.386** | **0.847** |

## ✅ 核心结论
1. **自研 RecursiveMaxMin 混合切分效果最优**
    - HitRate@3、MRR 双第一
    - 块集中度更高、长度适配模型、语义连贯性强
2. **外科学文本切分原则**
    - 长度 > 句子完整度
    - 语义一致性 > 结构规整度
3. **传统语义切分不适用于医学长文本**
    - 过度切分 → 知识点碎片化 → 召回率暴跌
4. **最佳RAG组合推荐**
    ```
    自研MaxMin混合切分 + bge-base-zh-v1.5 + 医学自适应命中规则
    ```
